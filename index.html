dsgNotepad
by Daniel Goodwin (@dsg on twitter if you need help)
Keyboard Shortcuts:
↵ Adds a typed in item to list.  
↑ Select the line above the current selection.  
shift ↑ Swaps current selected item with item above it.  
↓ Select the line below the current selection.  
shift ↓ Swaps current selected item with item below it.  
shift ← Decreases indent level.  
option ← Decreases completeness. If item is grayed out, the next decrease will delete it.  
shift → Increases indent level.  
option → Increases completeness.  
esc When nothing selected, makes current line editable. When something selected, disables editing of current line.  
Commands: The input area accepts commands. Commands are preceeded with a "\".
\h, \help Toggles this help area.  
\tweet Will take you to twitter with the current selection set to be tweeted.
